# 上海交通大学开题答辩 PPT 模板

两份 16:9 的上海交通大学开题答辩 PPT 模板，适合论文开题、课题汇报、阶段性研究汇报等场景。

## 下载

- [通用红色版](https://github.com/42shy6wt7y-hash/sjtu-proposal-defense-ppt-template/releases/download/v1.1.0/sjtu-proposal-defense-ppt-template.pptx)
- [深蓝版](https://github.com/42shy6wt7y-hash/sjtu-proposal-defense-ppt-template/releases/download/v1.1.0/sjtu-proposal-defense-ppt-template-deep-blue.pptx)

## 内容结构

- 封面
- 汇报提纲
- 研究背景与问题提出
- 文献综述与研究现状
- 研究目标、内容与创新点
- 研究方法与技术路线
- 可行性分析与风险控制
- 研究计划与进度安排
- 预期成果与可能贡献
- 致谢
- 标识来源与使用说明

## 标识来源

模板中使用的校徽和中英文横版标志来自上海交通大学官方视觉形象识别系统：

- [上海交通大学视觉形象识别系统 - 常用文件下载](https://vi.sjtu.edu.cn/index.php/articles/bulletin/13)
- [上海交通大学视觉形象识别系统管理办法](https://vi.sjtu.edu.cn/index.php/articles/bulletin/16)

请勿自行拉伸、重绘、改色或拼接校徽/校名标志。如需其他格式或最新规范，应以学校官方视觉形象识别系统发布内容为准。

## 重新生成

仓库中保留了生成脚本和官方来源素材。需要重新生成 PPTX 时运行：

```bash
python generate_sjtu_proposal_template.py
python generate_sjtu_proposal_template_deep_blue.py
```

生成文件为：

- `上海交通大学_开题答辩PPT模板.pptx`
- `上海交通大学_开题答辩PPT模板_深蓝版.pptx`
